# TUMUGI Mixer Module with Rust

TUMUGIのMixerモジュールは、Rustで実装された高性能な音声処理エンジンです。複数のDiscordサーバーを1つのUnionとしてまとめ、VC音声を相互接続することができます。MixerはOrchestratorからの指示に従って音声を中継し、参加サーバー間で同じVC体験を共有できるように設計されています。
Mixerは複数のUnionを同時に処理でき、各Unionは独立して動作します。また、複数のMixerインスタンスを配置することで、ユニオン単位で負荷分散や冗長性の確保が可能です。
Mixerは、音声の中継に特化しており、VC内での音声品質を維持しながら、低遅延で音声を転送します。これにより、参加サーバー間でシームレスなVC体験を提供します。
Mixerは、Orchestratorからの指示に基づいて、VC音声の中継を行います。Orchestratorは、Unionの状態を監視し、必要に応じてMixerに指示を送ります。これにより、Unionの参加サーバーの増減やVCの状態変化に柔軟に対応できます。
